구현 기능

설계부터 배포까지 완성된 기능 목록

🎨
AI 텍스타일 생성
텍스트 프롬프트로 8가지 스타일의 텍스타일 패턴 생성. SSE 실시간 진행률 스트림, seed 기반 재현, 배치(4개 동시) 지원
📤
이미지 업로드 & 자동 태깅
드래그앤드롭 업로드, AI가 텍스타일 어휘로 태그 자동 생성. 파일 타입/용량 검증, k3s PVC 스토리지
🗂️
컬렉션 & 즐겨찾기
디자인을 컬렉션으로 분류, 즐겨찾기 토글, 공개/비공개 설정, 공개 갤러리 페이지
🔐
인증 & 보안
JWT + Passport, bcrypt 패스워드, Next.js 미들웨어 쿠키 인증 가드, Helmet 보안 헤더, Rate Limiting
🔍
검색 & 필터
텍스트 검색, 스타일/소스/상태 필터, 페이지네이션. 대시보드 통계 (총 생성, 월별, 상태별)
🚀
CI/CD & k3s 배포
GitHub Actions 자동화, HPA(자동 스케일링), PDB(무중단 배포), Traefik Ingress, Kustomize
